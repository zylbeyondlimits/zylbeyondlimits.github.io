题目大意：

有一个的矩阵，满足第行第列的元素值为，求第大的值。

数据范围：

思路：

考虑二分答案。

假设我们现在枚举出了一个值为。

我们考虑到每一行比这个小的个数，要么是（当前行最大的数为），要么是。

直接统计答案即可。

复杂度：。

代码：

include typedef long long ll; const ll lim = 250000000000; using namespace std;

ll n, m, k, ans; bool check(ll x) {
ll res \= 0; for(ll i = 1; i <= n; i++)
res += min((x - 1) / i, m); return res < k;
} int main() {
cin >> n >> m >> k;
ll l \= 0, r = lim; while(l <= r) {
ll mid \= (l + r) >> 1; if(check(mid)) {
ans \= mid;
l \= mid + 1;
} else r \= mid - 1;
}
cout << ans << endl; return 0;
}
