：推出矩阵式子后用矩阵快速幂来维护。

代码：

ll ans \= 1; while(y) { if(y & 1)

x \= (ll)(x % mod * x % mod) % mod;
y >>= 1;
} return ans % mod;
} int main() {

}

T3:

链接：https://www.luogu.org/problem/T92569

：由于的数据范围很小，考虑枚举出的所有情况。有个优化即为把与输入重复的去除。先异或，然后用数出不同的个数，统计即可。

题目中最坏情况如何统计？即先取最小值后取最大值。

：多源最短路，即可。

代码：

include

o \= e - ‘0’;
p |= (o << j);
} if(!vis[p]) {
b[++cnt1] = p;
vis[p] \= true;
}
} for(int i = 0; i < (1 << k); i++) { if(!vis[i])
c[++cnt] = i;
} if(cnt == 0) {
cout << “0” << endl; return 0;
} for(int i = 1; i <= cnt; i++) { for(int j = 1; j <= cnt1; j++) { int res = b[j] ^ c[i]; while(res > 0) {

}

}
